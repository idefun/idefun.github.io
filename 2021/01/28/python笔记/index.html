python2与python3的区别

python2中的input与python3的不同,python2中input将从键盘得到的数据当成代码执行;python2中的raw_input()类似python3(没有raw_input())中为input() ,所以python2中可用raw_input()

注意：raw_input()和input()都获取的是字符串,字符串不能做-,*,/运算,做+运算只是拼接,可用int()函数获得数字,如int(input())
